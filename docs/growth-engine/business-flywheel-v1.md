# Business Flywheel v1.0

Status: Growth Engine Sprint 1  
Brand: MyKinLegacy  
Purpose: Define the complete business flywheel connecting discovery, trust, tools, collections, revenue, sharing, brand, and repeat growth.

This is the most important Growth Engine document in Sprint 1.

## 1. Master Flywheel

```text
Google
  -> AI
    -> Pinterest
      -> Guide
        -> Tool
          -> Collection
            -> Premium
              -> Subscription
                -> Brand
                  -> Google
```

The flywheel is not a linear funnel. Each layer strengthens the others.

## 2. Growth Flywheel

### Input

- human life moments
- search intent
- AI questions
- Pinterest visual demand
- family gift needs

### Output

- qualified visitors
- email leads
- interview starts
- paid orders
- family sharing

### How It Strengthens

Every strong collection creates:

- better examples
- better language
- better proof
- more sharing
- more branded search

### How It Auto-Grows

When families share collections, new family members discover the brand without ads.

## 3. Revenue Flywheel

## 4. Referral Flywheel

```text
Customer receives collection
```

### Output

The more a collection feels like "us," the more likely it is to be shared.

## 8. Channel Interaction

| Channel | Feeds | Receives from |
| --- | --- | --- |
| Google | Guides, tools, commercial pages | Brand search, customer language |
| AI | Definitions, FAQs, frameworks | Google authority, structured clarity |
| Pinterest | Visual meaning, gift ideas | Collection examples |
| Guide | Education and trust | Customer questions |
| Tool | Active engagement | Guides and AI questions |
| Collection | Revenue and proof | Tools, guides, referrals |
| Premium | Higher value | Core collection satisfaction |
| Subscription | Retention | Repeat life moments |
| Brand | Memory and trust | All successful experiences |

## 9. Failure Points

The flywheel breaks if:

- traffic is unqualified
- offer is unclear
- interview feels like a form
- collection feels generic
- AI is the hero
- sharing feels unsafe
- no one returns
- content is thin
- Pinterest is visual but meaningless
- Google traffic has no trust path
- AI systems cannot understand the category

## 11. Strategic Conclusion

The business flywheel starts with meaning and ends with meaning.

Google helps people find the question.

AI helps them understand the answer.

Pinterest helps them imagine the artifact.

Guides build trust.

Tools create engagement.

Collections create value.

Premium deepens value.

Subscription preserves value.

Brand remembers value.

And brand memory brings the next family back to Google, AI, Pinterest, or direct search.

The flywheel works only if the collection is worth sharing and preserving.
