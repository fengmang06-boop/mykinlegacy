# MyKinLegacy Crest Prompt Library v1

Status: concept direction only. Not integrated into production order fulfillment.

## Purpose

This library resets the crest artwork direction away from procedural line-art variation and toward reference-quality, high-fidelity concept generation.

The first approved test family is:

- Style family: Classic Shield Tree Crest
- Core theme: continuity and unity
- Core symbols: shield, tree, knot
- Visual world: black archival ground, antique gold embossed metal, private legacy object

## Directional Standard

The crest must feel like a premium family keepsake discovered in a private archive, not an illustration generated from simple geometry.

Required qualities:

- Embossed or engraved metal depth
- Realistic antique gold material
- Strong shield silhouette
- Central tree of life with roots
- Knot or woven root structure expressing unity
- Private archive / heirloom mood
- No official heraldry claim
- No legal coat of arms implication
- No customer-specific text in the image

## Master Prompt Pattern

```text
Use case: stylized-concept
Asset type: internal concept image for MyKinLegacy crest direction review
Input image: Use the attached CSO reference image as style inspiration only: luxury black background, antique gold embossed family crest objects, premium private archive mood.
Primary request: [CONCEPT NAME]. A luxury black and antique gold family legacy crest, [FRAME / SHIELD DESCRIPTION], central tree of life with visible roots, [KNOT DESCRIPTION], [LAUREL / PLAQUE / ORNAMENT DESCRIPTION], raised metallic gold relief, engraved bevels, dark archival leather background, high-end keepsake product.
Style/medium: high-fidelity raster concept, embossed antique metal, engraved details, premium product artwork.
Composition/framing: square 1:1, crest centered, full emblem visible, generous padding, strong silhouette.
Lighting/mood: warm gold highlights, soft shadows, deep vignette, ceremonial private archive feeling.
Constraints: no text, no watermark, no fake royal coat of arms claim, no official heraldry marks, no people, no logos.
Avoid: flat vector, simple line art, childish drawing, sci-fi, game logo, random geometry, clipart, low detail, thin outline, mechanical mask, disconnected circles, meaningless bars.
```

## Negative Prompt Library

Always avoid:

- Flat vector
- Thin outline
- Simple line art
- Childish drawing
- Coloring-book look
- Clipart
- Game clan logo
- Sci-fi shield
- Mechanical mask
- Random geometry
- Disconnected circles
- Meaningless bars
- Fake heraldic legal language
- Royal/noble title implication
- Text generated by the image model
- Watermarks

## MVP Text Rule

The image model must not generate family name, motto, or certificate text.

Text belongs to the server-side compositor for PDFs, certificates, posters, social cards, and future printable formats.
