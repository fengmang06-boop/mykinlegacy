# MyKinLegacy Crest Artwork V1 Design Spec

## Template

Template name: Shield Legacy Crest

## Visual Target

The crest should feel premium, private, archival, symbolic, family-oriented, gift-ready, calm, and dignified.

It should not feel random, mechanical, sci-fi, like a game logo, like a generic AI logo, like fake official heraldry, noisy, or confusing.

## Structure

- Outer shield frame as the protective family container.
- Inner gold border to create an archival, framed artifact feeling.
- Central tree as the dominant symbol for continuity and rooted legacy.
- Root knot or ring near the base to represent unity and connection.
- Subtle laurel or branch support marks that remain secondary.
- Bottom plaque area for visual grounding without embedded text.
- Dark archive background for non-transparent variants.
- Transparent output keeps only the shield/emblem artwork.

## Symbol Mapping

- `shield`: outer shield frame and protective border.
- `tree`: central trunk, branches, canopy, and roots.
- `knot`: intertwined ring/root knot at the lower center.
- `continuity`: circular ring, roots, balanced vertical axis.
- `unity`: knot, joined branches, connected lower plaque.

## Variant Rules

- Variant 1: full shield crest on dark archive background.
- Variant 2: closer emblem version, simplified and slightly larger.
- Variant 3: print-friendly version with stronger contrast.
- Transparent: clean shield/emblem on transparent background.

## Guardrail

This is deterministic MVP artwork for internal beta review. It must be technically valid and symbolically legible, but it must not be represented as official heraldry or final AI-generated artwork.
