# EXP-002: Values Validation

## Experiment ID

EXP-002

## Hypothesis

Family Values are more important than surname for Family Recognition because values define what the family stands for, while surname only identifies the family label.

## Variables

| Type | Description |
| --- | --- |
| Control | Surname + heritage country + animal + style |
| Test | Same inputs plus 3-5 Core Values |
| Fixed | Life moment omitted to isolate values effect |
| Changed | Addition of Core Values |

## Control

Surname-led interpretation.

## Method

Five controlled scenarios were compared. Each scenario used the same surname-style-symbol base. The test version added Core Values and observed changes in Narrative, Prompt, and Art Direction.

## Trials

| Trial | Base input | Added values | Control observation | Test observation |
| --- | --- | --- | --- | --- |
| V-001 | Surname Bennett, English, stag, green/gold | loyalty, humility, quiet strength | Stag looked noble but generic | Stag became gentle guardian; art direction softened |
| V-002 | Surname Marino, Italian, lion, blue/gold | resilience, service, family unity | Lion felt generic and royal | Lion became protective service symbol; less status-driven |
| V-003 | Surname Walsh, Irish, oak, deep green | endurance, humor, memory | Oak was standard heritage motif | Oak connected to memory and family humor |
| V-004 | Surname Novak, Czech, eagle, black/silver | independence, courage, migration | Eagle risked national-symbol feel | Eagle became migration and self-reliance symbol |
| V-005 | Surname Carter, American, bear, burgundy | protection, warmth, responsibility | Bear felt heavy and masculine | Bear became warm household protector |

## Evaluation

| Metric | Control Avg | Test Avg | Observation |
| --- | ---: | ---: | --- |
| Family Recognition | 5.7 | 7.8 | Values made identity more personal |
| Meaning Consistency | 5.8 | 8.2 | Values clarified symbol meaning |
| Narrative Consistency | 5.4 | 7.9 | Narrative gained direction |
| Gift Worthiness | 5.5 | 7.4 | Better emotional framing |
| Emotional Resonance | 5.3 | 7.6 | Values created warmth |
| Keepsake Value | 5.8 | 7.5 | More worth preserving |
| Authenticity | 5.9 | 7.7 | Less fake-heraldic |
| Story Depth | 5.2 | 7.6 | More interpretive depth |
| Identity Strength | 6.0 | 8.0 | Strongest identity signal |
| This Feels Like Us | 5.6 | 7.7 | Better recognition likelihood |

## Expected Result

Values should improve identity interpretation more than surname alone.

## Actual Observation

The hypothesis was supported. Surname gave a label; values gave the artifact a personality. The surname remained useful for personalization, but it did not explain why the family should be represented in a particular way.

## What We Learned

Core Values should be treated as a primary interpretation layer. Surname should not be treated as the main identity source.

Yes.
