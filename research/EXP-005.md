# EXP-005: Identity DNA Validation

## Experiment ID

EXP-005

## Hypothesis

Identity DNA changes interpretation quality because it captures deeper family patterns that raw inputs cannot express.

## Variables

| Type | Description |
| --- | --- |
| Control | Raw inputs only |
| Test A | Raw inputs + partial Identity DNA |
| Test B | Raw inputs + full Identity DNA |
| Fixed | Same family scenario and artifact type |
| Changed | Depth of Identity DNA |

## Control

Raw inputs:

- surname
- country
- animal
- colors
- motto
- style

## Method

Three scenarios were evaluated across three depth levels:

- Control: raw inputs only
- Test A: partial DNA with Core Values, Emotional Tone, Legacy Theme
- Test B: full DNA with values, promise, energy, conflict, strength, weakness, future hope, tradition, purpose, recognition signal

## Trials

| Scenario | Control observation | Partial DNA observation | Full DNA observation |
| --- | --- | --- | --- |
| IDNA-001 Father gift | Strong-looking crest but generic | Warm protector theme emerged | Quiet labor, safety, gratitude, and father-recognition signal became central |
| IDNA-002 Blended family | Confusing surname identity | Shared values clarified unity | Inclusion, chosen belonging, new household promise, and avoidance of bloodline language emerged |
| IDNA-003 New grandparent | Legacy crest with baby tone | Future hope appeared | Grandparent-to-grandchild wisdom, continuity, humility, and letter artifact direction became clear |

## Evaluation

| --- | ---: | ---: | ---: | --- |
| Family Recognition | 5.9 | 7.2 | 8.2 | DNA improved specificity |
| Meaning Consistency | 6.0 | 7.4 | 8.3 | Values and promise clarified meaning |
| Narrative Consistency | 5.6 | 7.5 | 8.5 | Full DNA gave narrative logic |

| Identity Strength | 6.2 | 7.5 | 8.4 | Clearer family pattern |
| This Feels Like Us | 5.8 | 7.2 | 8.1 | Recognition improved |

## Expected Result

Identity DNA should change final interpretation in meaningful ways.

## Actual Observation

The hypothesis was supported, but with a caution: full Identity DNA produced the strongest interpretation but may require more input and more care. Partial DNA already improved outcomes. The optimal production path may need progressive DNA collection rather than forcing all fields upfront.

## What We Learned

Identity DNA is valuable, but input burden matters. The most useful early DNA fields were Core Values, Emotional Tone, Legacy Theme, Future Hope, and Recognition Signal.

## Should Continue?

Yes, with caution.

## Confidence

Low-Medium. The sample size is smaller than EXP-001 and the method is more interpretive.

## Next Experiment

EXP-010 should test which Identity DNA fields create the most recognition lift with the least customer effort.
