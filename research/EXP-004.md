# EXP-004: Meaning Graph Validation

## Experiment ID

EXP-004

## Hypothesis

A Meaning Graph is more important than a simple symbol list because it connects symbols to people, values, emotions, and legacy.

## Variables

| Type | Description |
| --- | --- |
| Control | Simple symbol meaning |
| Test | Connected Meaning Graph |
| Fixed | Same symbol, family scenario, and intended artifact type |
| Changed | Depth and relationship of meaning |

## Control

Simple symbol:

"Oak means strength."

## Method

Five symbols were tested as simple meanings and then as Meaning Graphs. The output difference was observed in prompt specificity and art direction.

## Meaning Graph Trials

| Trial | Control simple symbol | Test Meaning Graph | Observation |
| --- | --- | --- | --- |
| MG-001 | Oak = strength | Oak -> protection -> grandfather -> responsibility -> legacy | Became memorial and generational |
| MG-002 | River = journey | River -> migration -> two homes -> belonging -> future children | Became immigrant-family specific |
| MG-003 | Bear = protection | Bear -> father -> quiet labor -> safety -> gratitude | Became father's gift specific |
| MG-004 | Laurel = honor | Laurel -> anniversary -> endurance -> shared promise -> celebration | Became couple milestone specific |
| MG-005 | Star = guidance | Star -> grandmother -> stories -> child learning roots -> continuity | Became child/grandparent bridge |

## Evaluation

| Metric | Control Avg | Test Avg | Observation |
| --- | ---: | ---: | --- |
| Family Recognition | 5.8 | 8.2 | Graph made symbols personal |
| Meaning Consistency | 6.0 | 8.8 | Strongest improvement |
| Narrative Consistency | 5.6 | 8.1 | Narrative emerged naturally |
| Gift Worthiness | 5.6 | 7.9 | Better recipient fit |
| Emotional Resonance | 5.5 | 8.1 | Symbol carried feeling |
| Keepsake Value | 5.7 | 8.0 | More worth preserving |
| Authenticity | 5.8 | 8.0 | Less dictionary-like |
| Story Depth | 5.2 | 8.5 | Major improvement |
| Identity Strength | 5.9 | 8.0 | Clearer identity |
| This Feels Like Us | 5.7 | 8.1 | Strong lift |

## Expected Result

Meaning Graph should produce more specific and recognizable interpretation.

## Actual Observation

The hypothesis was supported. Simple symbols created generic meaning. Meaning Graphs created family-context meaning. The biggest improvement was in Meaning Consistency and Story Depth.

## What We Learned

A symbol should not enter art direction unless it has a meaning path. The path should connect at least one symbol, one value, one human anchor, and one legacy idea.

Yes.
